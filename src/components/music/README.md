# Background Music

Add your background music files here.

## Instructions

1. Place your audio file(s) here (.mp3, .wav, .ogg, etc.)
2. Supported formats: mp3, wav, ogg, webm, aac
3. The music will play in the background when the site loads

## Setup

To use background music:

1. Add your music file to this folder (e.g., background.mp3)
2. Update the music player component to load from this folder (or edit via the MusicTab)

## Recommended

- Use royalty-free music
- Keep file sizes reasonable (under 5MB for best performance)
- Use MP3 format for best browser compatibility
